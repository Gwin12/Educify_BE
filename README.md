# Educify Backend

This is the backend API for the Educify Recommendation Page built with **Node.js**, **Express**, and **TypeScript**.

---

## Features
- Generate a teacher's share link.
- Send share link via email.
- Global error handling middleware.
- Fully typed with TypeScript.
